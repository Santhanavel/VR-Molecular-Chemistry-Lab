5. Reset Flow
Players must be able to break a formed molecule and reclaim its atoms. Reset can be triggered two ways: physically pulling a molecule apart (distance threshold), or pressing the Reset button on the Inspector Panel.
5.1  Reset Steps
Player grabs the molecule object (XRGrabInteractable on the molecule prefab root).
Player moves the molecule more than resetDistanceThreshold (0.3m) from its spawn point, OR presses the UI Reset button.
BondManager.ResetMolecule(GameObject mol) is called.
All AtomController references stored on the molecule component have ResetAtom() called.
Each atom re-enables its renderer and collider; positions are reset to AtomTable spawn points.
The molecule prefab is Destroyed.
BondManager fires OnMoleculeReset; UIManager updates the Library Panel (molecule remains in discovered list but is no longer "formed").
5.2  MoleculeInstance Component
A lightweight MonoBehaviour on every molecule prefab that stores the atom list and exposes the reset API:
public class MoleculeInstance : MonoBehaviour
{
    public MoleculeData Data;
    public List<AtomController> ConsumedAtoms;
    // Called by BondManager
    public void Dissolve()
    {
        foreach (var atom in ConsumedAtoms)
            atom.ResetAtom();
        Destroy(gameObject);
    }
}
