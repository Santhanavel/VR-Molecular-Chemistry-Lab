2. ScriptableObject Population
2.1  Manual Inspector Method (Recommended for first pass)
In the Unity Project window, navigate to Assets/_Project/ScriptableObjects/
Select MoleculeDatabase.asset
In the Inspector, click the + button on the Molecules list
Fill each field from the table in Section 1
Drag the corresponding molecule prefab into the moleculePrefab slot
Repeat for all 18 entries
2.2  JSON Import Script (Optional speedup)
To avoid 18 ГѓвЂ” manual data entry, create a Unity Editor script that reads a JSON file and populates the ScriptableObject programmatically. Run it once from the Unity menu.
// Save as: Assets/_Project/Editor/MoleculeDatabaseImporter.cs
[MenuItem("VRLab/Import Molecule Database")]
static void ImportDatabase()
{
    var db = AssetDatabase.LoadAssetAtPath<MoleculeDatabase>(
        "Assets/_Project/ScriptableObjects/MoleculeDatabase.asset");
    var json = File.ReadAllText("Assets/_Project/Data/molecules.json");
    var data = JsonUtility.FromJson<MoleculeDataList>(json);
    db.molecules = data.items;
    EditorUtility.SetDirty(db);
    AssetDatabase.SaveAssets();
    Debug.Log("Imported " + db.molecules.Count + " molecules.");
}

3.1  Bond Cylinder Sizing
// Utility called at prefab build time (or runtime spawn)
void CreateBondCylinder(Transform from, Transform to, BondType type)
{
    var mid   = (from.position + to.position) * 0.5f;
    var dir   = to.position - from.position;
    float len = dir.magnitude;
    var cyl   = GameObject.CreatePrimitive(PrimitiveType.Cylinder);
    cyl.transform.position   = mid;
    cyl.transform.up         = dir.normalized;
    cyl.transform.localScale = new Vector3(0.015f, len * 0.5f, 0.015f);
    // Double bond: spawn two offset cylinders side by side
    // Triple bond: spawn three cylinders in a triangle pattern
}
4. Android / Meta Quest Build Pipeline
4.1  One-Time Setup Steps
Install Android Build Support module via Unity Hub > Installs > Add Modules.
Install Android SDK & NDK Tools (bundled with Unity Hub Android module).
In Unity: Edit > Project Settings > Player > Android tab:
Company Name: your name; Product Name: VRMolecularLab
Set Package Name: com.yourname.vrmolecularlab
Set Minimum API Level: Android 10 (API 29)
Set Target API Level: Android 13 (API 33)
Scripting Backend: IL2CPP
Target Architectures: ARM64 only (uncheck x86, x86_64)
In XR Plug-in Management > Android tab: enable OpenXR.
In OpenXR > Features: enable Meta Quest Support and Hand Tracking Subsystem.
Create a debug keystore: Player Settings > Publishing Settings > Keystore Manager.

5. Quest Optimisation Checklist
Optimisation
Implementation detail
GPU Instancing
Enable on all atom and molecule materials; allows all same-element atoms to batch in one draw call
Static batching
Mark LabRoom, AtomTable, walls, floor as Static; Unity batches them at build time
LOD Groups
Lab environment mesh: LOD0 (full) at <3m, LOD1 (50% tri) at >3m; none needed on atoms (already low-poly)
Occlusion Culling
Bake via Window > Rendering > Occlusion Culling; small lab means minimal savings but costs nothing
Texture atlasing
Combine all UI sprites into a single 1024x1024 atlas; reduces UI draw calls from N to 1
Audio compression
Bond SFX: Vorbis, 44kHz, Load Type = Decompress on Load; Ambient: Vorbis, 22kHz, Streaming
URP Forward+
Use URP Forward+ renderer; disable HDR; disable MSAA (Quest uses FXAA); disable post-process
Physics layers
Matrix: Atom-Atom = collide; Atom-UI = none; Atom-Environment = collide; reduces OverlapSphere cost
Profiler testing
Connect Quest via Quest Link; Window > Analysis > Profiler; target CPU < 10ms/frame, GPU < 11ms/frame
