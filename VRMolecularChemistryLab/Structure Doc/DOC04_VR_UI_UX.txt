Overview
This document defines every UI panel in the application â€” what it shows, how it is positioned in the world, how the player interacts with it via ray-cast, and how it is implemented in Unity. All UI is world-space (no screen-space overlays). All text uses TextMeshPro.
XR UI golden rule
Never place UI elements closer than 0.8m or farther than 3m from the player. The ideal reading distance is 1.2mâ€“1.8m. Panel text must be at least 14pt equivalent at 1.5m viewing distance. High-contrast colours only (WCAG AA minimum).

2.2  Layout
Panel header: "Molecule Library" in 0.05m TMP bold, white on navy strip.
Below header: scrollable grid of molecule cards. 2 columns x N rows. Each card shows:

Bond type badge: SINGLE / DOUBLE / TRIPLE (coloured pill)
Undiscovered slots: greyed-out silhouette card with "???" label. Fills in with DOTween fade+scale when discovered.

Unity ScrollRect component; Vertical scroll only; Scroll Sensitivity = 0.03 (VR-friendly slow scroll); Scrollbar hidden (swipe with ray to scroll).

public void AddDiscovery(MoleculeData mol)
{
    if (_discovered.Contains(mol.moleculeName)) return;
    _discovered.Add(mol.moleculeName);
    var card = Instantiate(moleculeCardPrefab, libraryGrid);
    card.GetComponent<MoleculeCard>().Populate(mol);
    // DOTween punch scale on new card
    card.transform.localScale = Vector3.zero;
    card.transform.DOScale(1f, 0.35f).SetEase(Ease.OutBack);
}

4. Wrist Menu
A quick-access panel anchored to the inside of the player's left wrist. Toggled by pressing the Menu button. Allows spawning additional atom tokens without walking back to the AtomTable.
4.1  Anchor Setup
// In UIManager.Start():
wristMenuCanvas.transform.SetParent(leftHandAnchor);
wristMenuCanvas.transform.localPosition = new Vector3(0, 0.05f, 0.05f);
wristMenuCanvas.transform.localRotation = Quaternion.Euler(45f, 0, 180f);
wristMenuCanvas.transform.localScale = Vector3.one * 0.0012f; // 1.2mm per canvas unit
4.2  Content
Four atom spawn buttons (H / O / C / N) each showing the element colour swatch and symbol. Pressing a button calls AtomSpawner.SpawnAtom(elementSymbol) which instantiates the corresponding atom prefab 0.3m in front of the player.
4.3  Show / Hide Animation
public void ToggleWristMenu()
{
    _wristMenuVisible = !_wristMenuVisible;
    float targetAlpha = _wristMenuVisible ? 1f : 0f;
    canvasGroup.DOFade(targetAlpha, 0.2f);
    float targetScale = _wristMenuVisible ? 1f : 0f;
    wristMenuCanvas.transform.DOScale(targetScale, 0.2f)
        .SetEase(_wristMenuVisible ? Ease.OutBack : Ease.InBack);
}
5. Interaction Ray Setup
Both controllers have two interactors attached: XRDirectInteractor (short-range grab) and XRRayInteractor (long-range UI pointer). The ray interactor is configured as follows:

Direct vs Ray interactor
Direct interactor is used for grabbing atoms (requires physical proximity). Ray interactor is for UI only. The two never conflict because atom colliders are on the "Atom" layer and UI panels are on the "UI" layer, and each interactor's Interaction Layer Mask includes only its own layer.
